##Prerequisites

- Bicep CLI
  - Please ensure the Bicep CLI has been installed on your local machine as per - https://learn.microsoft.com/en-us/azure/azure-resource-manager/bicep/install

- Azure PowerShell
  - Please ensure that the most recent version of Azure PowerShell has been installed as per 
    ```powershell
    Install-Module Az
    ```
    or
    ```powershell
    Update-Module Az
    ```
